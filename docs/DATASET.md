# Dataset Access

QadamCare AI was developed using the Foot Ulcer Segmentation Challenge dataset for RGB diabetic-foot ulcer segmentation research.

The dataset is not included in this repository because it contains medical images and may have licensing, privacy, and redistribution restrictions.

Users who want to reproduce the training pipeline should obtain the dataset from the official dataset source and place it locally using the expected project structure.
